# ERA — Luxury Event Booking Homepage

A modern, luxury-themed event booking homepage for the **ERA** brand.  
Built with **React (Vite)** + **TailwindCSS**.

---

## 🚀 Tech Stack
- React 18 + Vite
- TailwindCSS 3
- PostCSS + Autoprefixer
- Google Fonts (Playfair Display & Inter)

---

## 🎨 Brand Color Palette
- **Primary Background**: `#0D0D0D` (Jet Black)
- **Text Primary**: `#FFFFFF` (White)
- **Accent / CTA**: `#D4AF37` (Champagne Gold)
- **Text Secondary**: `#B0B0B0` (Soft Gray)
- **Border**: `#1F1F1F` (Dark Gray)
- **Star (ratings)**: `#F5C518` (Warm Yellow-Gold)
- **Alt Mobile Background**: `#FAF8F5` (Cream)
